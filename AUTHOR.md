# Project Author

Name: Shekhar Pratyush

Project: LeetCode Heatmap Widget
Year: 2026

Description:
This project is an Android widget that displays a GitHub-style heatmap
showing the last 30 days of LeetCode activity for a user.

The widget fetches data using the LeetCode GraphQL API and renders
the submission activity directly on the Android home screen.

Features:

* 30-day activity heatmap
* Clickable widget opening the activity dashboard
* Username configuration
* GraphQL API integration

Created by Pratyush as a personal learning project.
